# Calendar Booking Continuation

This repository continues the completed calendar booking project. The
application code is copied from the previous project; the current project
focuses on organizing agent-assisted GitHub workflow around issues, triage,
pull requests, review iterations, and scheduled checks.

Further development ideas are listed in `docs/development-plan.md`.

## Production deploy

Docker build is configured in `Dockerfile`. The container starts the app automatically and listens on `PORT`.

Render deployment is described in `render.yaml`.
